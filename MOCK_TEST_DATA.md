# 🧪 Recommended Testing Order

Test in this order for maximum coverage of different scenarios:

| # | Test Case | What It Tests |
|---|-----------|---------------|
| 1 | **Howdy Burgers** (#6) | Service + food + logo + product image |
| 2 | **Nabila's Salon** (#5) | Service + logo only (NO product image) |
| 3 | **InvoiceHero** (#8) | Digital product + logo only (SaaS) |
| 4 | **MoneyMindset** (#12) | NO logo + NO product image (hardest case!) |
| 5 | **Khaadi** (#2) | Fashion + full assets + portrait ratio |
| 6 | **Audionic** (#3) | Tech/Electronics + full assets |
| 7 | **SkillUp Academy** (#7) | Education + landscape ratio |
| 8 | **CareFirst Dental** (#11) | Healthcare service |
| 9 | **Zameen Developers** (#10) | Real estate + premium feel |
| 10 | **Medora** (#4) | Beauty/cosmetics |
| 11 | **FitTrack** (#9) | Mobile app + story ratio |

### Why This Order?
- **#1-4** cover the 4 major scenarios (physical+assets, service+logo, digital+logo, nothing)
- **#5-11** cover different industries and aspect ratios
- If #1-4 all look good, system is solid across all categories
